# Clinic-by-C
A clinic management system by C
There Are two modes (Admin Mode, User Mode)
    A- Admin mode you can do the follow:
        1-Adding infinite number of patients (by malloc)
        2-Adding Patient info (Name,Age,ID,Gender) (in struct) using functions
        3-ID is unique and can not be duplicated
        4-you can edit any patient info anytime 
        5-Adding reservation from 5 available slots
    B- In User Mode you can do the followin
        By Entering patient ID you can read the follow
        1-Patient Name
        2-Patient Age
        3-Patient Gender
        4-Slots the patient had reserved before
